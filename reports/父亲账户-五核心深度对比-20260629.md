## 二、逐家分析

### 2.1 恒瑞医药 600276 —— 父亲账户最高质量标的，但估值不便宜

**赚什么钱**：创新药销售（2025年163亿，占58%）+ 仿制药（集采承压但占比下降）+ 海外BD许可费（34亿）

**与父亲方向匹配度**：★★★★★ 直接对应"创新药"方向，组合中最纯正的标的

**财务质量**：
- 2025年营收316亿(+13%) / 净利77亿(+22%) / 毛利率86.2%
- 2026Q1净利22.8亿(+22%)，利润增速持续快于收入，规模效应显现
- 研发投入87亿(27.6%)，在研管线全球第二
- 创新药收入占比62%（2026Q1），已完全取代仿制药成为主引擎
- 负债率极低，现金充沛，几乎零财务风险

**护城河**：★★★★★
- 品牌/渠道：19个创新药上市，覆盖中国90%+大型医院
- 技术壁垒：研发87亿/年 + 5,600人团队 + 全球第二大管线
- 专利：仿制药集采已基本消化，创新药有10-15年专利保护
- 平台：小分子+大分子+ADC+CGT全模态覆盖

**估值**：PE 42.3x
- 处于自身历史30-70x区间的**中位偏下**
- 利润增速22% → PEG ~1.9x，偏贵但不算离谱
- 如果创新药增速达到公司目标30%+，PEG可降到1.4x

**最大风险**：
1. 集采继续扩面——仿制药收入（42%）进一步承压
2. 出海BD节奏——如果对外许可断档，34亿许可费收入可能波动
3. 研发回报率——如果III期临床失败，87亿投入可能打折扣
4. 估值压缩——如果创新药增速不达30%，PE可能杀到30-35x

**段永平式追问**：恒瑞的生意本质是"用研发赌专利保护期内的垄断利润"。这门生意的关键在于——研发效率是否高于同行？恒瑞2019-2025年19个创新药上市，全球第二。研发效率很高。但PE 42x意味着市场已经给了不低的溢价。

**家庭账户适配度**：★★★★ 核心仓候选。唯一扣分是当前估值不便宜，安全边际不足。

**融资适配度**：**谨慎**。恒瑞财务安全性极高（几乎没有有息负债），在上海主板不是科创/创业板，波动相对可控。但PE 42x意味着如果创新药增速不达预期，可能面临30%+的估值压缩。融资会放大这个风险。

**在父亲账户中建议角色**：**核心仓候选**——保留研究，等待PE回到35-38x区间再加重仓。

### 2.2 许继电气 000400 —— 五只中最适合家庭账户融资持有的标的

**赚什么钱**：电网二次设备（继电保护/变电站自动化）+ 特高压直流换流阀 + 智能电表 + 充电桩

**与父亲方向匹配度**：★★★★★ 直接对应"电力"方向。电网二次设备+特高压与之前在电力漏斗中筛选的国电南瑞属于同一赛道。

**财务质量**：
- 2025年营收150亿(-12.3%)——主动压降低毛利EPC业务+特高压交付延迟
- 2025年净利11.7亿(+4.5%)——增利不增收，结构优化中
- 2026Q1净利1.11亿(-46.5%)——降价电表订单交付拖累
- 经营现金流26.7亿(+106%)，远大于净利润，**利润质量优秀**
- 毛利率逐板块提升（除智能电表），综合毛利率+2.6pct至23.4%
- 国际业务营收+73%，毛利率26.9%

**核心增长变量**：
- **特高压直流在手订单约50亿**，预计2026H2开始集中交付，直流输电板块营收有望翻倍至24亿+
- "十五五"电网投资4万亿（较十四五+40%），新核准"四直三交"
- 国际业务（沙特/巴西/西班牙）加速
- **关键拐点预计在2026Q3**——降价电表订单消化完毕+特高压大单开始确认

**护城河**：★★★★
- 技术壁垒：特高压换流阀市占>40%，全球唯三能做柔直换流阀的企业
- 客户锁定：国网/南网采购体系，供应商认证周期长
- 牌照/资源：央企背景（国家电网旗下），天然准入优势

**估值**：PE 25.0x
- 2026E PE ~22x（按券商一致预测净利13.7-14.5亿）
- 2027E PE ~16x（按18-20亿）
- 对于一家受电网投资超级周期驱动、经营现金流远超净利的公司，**PE 25x不算贵**

**最大风险**：
1. 特高压项目交付节奏不确定——如果国网排程延迟，业绩拐点可能推后
2. 国网集采降价压力——智能电表/配网招标价格下行
3. Q1利润腰斩的短期压力——需要Q3验证拐点
4. 国电南瑞是更强竞争标的——如果在电力方向只选一只，南瑞的确定性更高

**段永平式追问**：许继的生意本质是"电网投资的收费桥梁"——只要电网投资增长，许继就受益。但这门生意的一个天然缺陷是对单一大客户（国网）的依赖。好在国网是央企，付款信用好，坏账风险低。

**家庭账户适配度**：★★★★★ 五只中最适合家庭账户长期跟踪。电网设备+特高压的确定性高，PE合理，有分红。

**融资适配度**：**适合**（五只中唯一）。理由：深主板标的，波动相对可控；PE 25x有安全边际；经营现金流远超净利，盈利质量真实；央企背景+电网投资确定性高。Q1业绩压力是短期扰动，非永久性恶化。

**在父亲账户中建议角色**：**核心仓候选**——可以继续跟踪，关注2026Q3特高压交付拐点。

### 2.3 天孚通信 300394 —— 好赛道好公司，但客户集中度是致命软肋

**赚什么钱**：光模块核心器件（光引擎/光器件），卖给Fabrinet→英伟达的光模块供应链

**与父亲方向匹配度**：★★★★ "算力/光通信"方向，是AI算力链条上最"硬"的环节之一

**财务质量**：
- 2025年营收51.6亿(+59%) / 净利20.2亿(+50%)，均创历史新高
- 毛利率54%（从57%下滑），产品降价+泰国工厂投产初期成本高
- 经营现金流18.7亿(+48%)
- 负债率极低，现金充沛

**致命软肋——客户集中度**：
- **第一大客户Fabrinet占营收63.3%**（2025年较2024年61.7%进一步上升）
- 前五大客户占89.7%
- 如果失去Fabrinet/英伟达订单——不是"利润下滑"的问题，是"生存危机"

**护城河**：★★★
- 技术壁垒有（光引擎精密制造），但不是不可逾越
- 客户关系深绑定，但**这种绑定是单向依赖**——天孚需要英伟达 >> 英伟达需要天孚
- 如果英伟达切换供应商，天孚几乎没有反制能力

**估值**：PE 31.5x（2025年净利），2026E PE ~25x（按32亿净利预测）。如果2026E PEG ~0.8x（净利增速30%-50%），估值合理。但这依赖于两个关键假设：(1)英伟达不换供应商；(2)AI算力资本开支不减速。

**最大风险**：
1. **客户集中度63%是第一大风险**——比任何财务指标都更重要
2. 英伟达CPO路线可能改变供应链格局
3. 1.6T上量节奏不及预期（机构预测的2026E净利可能偏高）
4. AI算力资本开支周期反转——AI投资一旦减速，光模块需求直接承压

**芒格式追问**：如果我错判了天孚通信，"错在哪"？最可能的答案是——英伟达更换供应商，或者AI资本开支周期见顶。前者导致估值从30x PE杀到15x，后者导致利润负增长+"戴维斯双杀"。**客户集中度63%意味着天孚不是一门"独立"的生意——它是英伟达产业链上的一个附属环节。**

**家庭账户适配度**：★★ 算力方向的好赛道，但客户集中度63%意味着这门生意缺乏独立性。适合**小仓位卫星观察**，不适合作为核心仓。

**融资适配度**：**不适合**。创业板+高客户集中度+AI周期不确定性。如果AI算力资本开支周期反转，戴维斯双杀的杀伤力会被融资放大2倍。

**在父亲账户中建议角色**：**卫星仓候选**——小仓位保留观察（不超过总资产3-5%），绝不能用融资。

### 2.4 中芯国际 688981 —— 战略价值最高，但财务质量最不适合家庭账户

**赚什么钱**：晶圆代工——给芯片设计公司（华为海思/紫光展锐等）制造芯片，赚取加工费

**与父亲方向匹配度**：★★★★ 半导体制造是算力链条的底层基础设施。但中芯国际更偏"半导体制造"，与父亲关注的"算力/物理AI"是间接关系。

**财务质量**：
- 2025年营收673亿(+16.5%) / 净利50.4亿(+36.3%)
- 毛利率21%（极低——对比台积电55%+）
- **FCF为负**（资本开支81亿美元/年远超经营现金流）
- 折旧逐年加重（2026年折旧同比+30%）
- ROE仅7.3%
- 2026Q1净利13.6亿(+0.4%)，利润几乎零增长——折旧吃掉所有增量

**核心矛盾**：中芯国际的"战略价值"和"财务价值"之间存在巨大鸿沟。
- 战略上——中国最大、最先进的晶圆代工厂，国产替代不可替代性。中国政府不可能让它倒下
- 财务上——永无止境的资本开支（每年80亿美元），低毛利率（21%），低ROE（7%），负自由现金流，零分红

**护城河**：★★★★★（战略级护城河，但不是经济护城河）
- 中国唯一能做14nm及以下制程的晶圆厂
- 资本壁垒：需要500亿+美元才能复制
- 人才壁垒：中国最资深的半导体制造团队
- 但护城河**没有转化为超额利润**——低毛利率和负FCF说明竞争地位没有转化为股东价值

**估值**：PE 55.6x（按2025年净利）
- 对于ROE 7.3%、FCF为负的公司，PE 55x在纯财务框架下无法解释
- 估值逻辑是"战略稀缺性溢价"——类似"和平时期的国防预算"
- 如果只看财务，合理的PE应该在20-25x（对应净利50亿，市值1000-1250亿）

**最大风险**：
1. 美国更严厉制裁——限制美系设备/材料出口
2. 资本开支黑洞——如果AI需求不如预期，80亿美元/年的投入可能产出极低
3. 先进制程被封锁——没有EUV，可能永远止步于7nm
4. 折旧持续侵蚀利润

**李录式追问**：20年后回看，中芯国际是"这个时代的台积电"还是"中国需要的但不赚钱的基础设施"？我认为更接近后者——它是中国的半导体战略资产，但它的经济特征像公共事业（低毛利/重资产/高资本开支），而不是像台积电那样的利润机器。

**家庭账户适配度**：★★★ 战略价值最高。但财务上不适合追求"睡得着觉"的家庭账户。更适合作为**国家战略主题的少量配置**，而非核心持仓。

**融资适配度**：**不适合**。科创板+高PE+负FCF+折旧暴增+地缘风险。五只中融资风险最高的一只。

**在父亲账户中建议角色**：**卫星仓候选**——如果父亲看重其战略稀缺性，可保留极小仓位（<3%），但绝不能用融资。作为换仓资金来源也是合理选择——卖掉中芯国际换恒瑞/许继/长江电力，财务质量立即改善。

### 2.5 澜起科技 688008 —— 生意最好但估值令人不安

**赚什么钱**：内存接口芯片（DDR5 RCD/MRCD/MDB），卖给三星/SK海力士/美光的内存模组厂。全球第一（市占37-43%）。

**与父亲方向匹配度**：★★★★ 直接受益AI服务器对高速内存的需求。属于"算力/AI芯片链"方向。

**财务质量**：
- 2025年营收54.6亿(+50%) / 净利22.4亿(+58%)
- 毛利率62.2%，2026Q1进一步升至69.8%（互连芯片毛利率71.5%）
- 经营现金流20.2亿，匹配净利
- 四项AI新品收入同比+93.8%，占互连芯片收入19%
- 负债率极低，现金充沛（2026Q1总资产217亿，含H股融资）

**护城河**：★★★★★
- 全球第一内存接口市占37-43%
- 与瑞萨/IDT/Rambus三寡头格局，新进入者几乎不可能
- JEDEC标准制定者（唯一中国公司进入董事会）
- DDR5子代每代升级都需要重新认证——**子代升级本身就是竞争壁垒**
- 轻薄短小的无晶圆厂模式（fabless），轻资产+高毛利

**估值**：PE 80.4x（按2025年净利），2026E PE ~45-50x（按一致预测33-35亿净利）
- 绝对值：非常贵
- 如果只看2026E PE 45-50x + 净利增速50% → PEG ~1.0x，刚刚合理
- 但2027年增速可能放缓到30%+ → 2027E PE ~35x → 估值逐步消化

**最大风险**：
1. **内存周期**——历史上澜起科技的内存接口芯片收入与全球DRAM出货量高度相关。如果内存周期下行，澜起的"AI故事"可能会被"周期下行"覆盖
2. DDR5渗透率见顶——目前65%，天花板可能是85-90%。剩余空间有限
3. 估值极高——PE 80x意味容错空间为零
4. 科创板波动+小盘成长特征——不适合融资

**段永平式追问**：澜起的生意本质是"内存接口的标准制定者"——这确实是个好生意。但PE 80x买入的问题是——你不仅要赌DDR5渗透率继续提升，还要赌AI服务器内存需求高增长，还要赌澜起维持全球第一市占率，还要赌内存周期不反转。四个"还要"叠加，需要的"对"太多了。

**家庭账户适配度**：★★ 生意最好但估值最贵。需要等待PE回到50-60x（对应2026E）或等待2027E PE降到30x以下。当前价格不适合重仓。

**融资适配度**：**不适合**。科创板+PE 80x+内存周期风险。即使在五只中生意最好，融资持有也是不合适的。

**在父亲账户中建议角色**：**卫星仓候选**——保留小仓位观察（<3%），等待估值压缩到合理区间。当前不建议加仓。

## 五、最终结论

### 5.1 如果只能保留3只继续深度跟踪

| 保留 | 剔除 | 理由 |
|------|------|------|
| **恒瑞医药** | — | 创新药方向唯一核心，财务最安全，与父亲方向完美匹配 |
| **许继电气** | — | 电力方向核心，PE最低，五只中唯一适合融资持有 |
| **天孚通信** | — | 算力方向代表，好赛道。保留但仅作小仓位卫星 |
| — | 中芯国际 | ROE 7%+FCF为负+零分红。不适合家庭长期持有。可换恒瑞/许继 |
| — | 澜起科技 | PE 80x太贵。等估值压缩后再研究。生意好≠现在买入 |

### 5.2 如果只能选1只作为最稳健核心候选：**许继电气 000400**

**理由**：
1. PE 25x是五只中最低的，有安全边际
2. 经营现金流26.7亿(2025)远超净利11.7亿——利润质量是真实的
3. 电网投资超级周期（十五五4万亿）提供确定性增长
4. 特高压在手订单50亿+2026H2集中交付=明确拐点
5. 央企背景+深主板+有分红——最接近"拿着能睡好觉"
6. **五只中唯一适合融资持有**

反面论据：Q1利润腰斩是短期扰动（降价电表交付），不是永久性恶化。如果Q3验证拐点/特高压开始批量确认，这个判断会得到验证。如果Q3继续低于预期，需要重新评估。

> 提醒：许继电气PE 25x vs 国电南瑞PE 26x。两者都是电网设备好标的。南瑞护城河更深（市占80%+），许继特高压弹性更大。如果只能选一只电力仓位，南瑞更稳健；如果追求弹性+当前价格合适，许继。两只可以互补配置。

### 5.3 值得继续研究、不应该被剔除的：全部5只都值得继续研究

但没有一只现在可以"闭眼买入"。具体的等待条件：
- 恒瑞医药：等PE降到35-38x 或 创新药增速确认提高到30%+
- 许继电气：等2026Q3验证特高压交付拐点，当前可小仓位建观察仓

- 澜起科技：等PE降到50-60x（对应2026E）

父亲账户当前最大问题是50只持仓+72%融资。**减量+降杠杆比选股更重要。**

如果只能做一件事：把融资比例从72%降到30%以下。在这个基础上再考虑这5只中哪些值得加仓。
